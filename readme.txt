できること

ボイスチャンネルAでbotが聞いた内容をボイスチャンネルBで別のbotがそのまま流す



準備

tokens.jsonに作っておいた2つのbotのトークンを配置しておく。
音声を聞く側はlistener、流す側はstreamerっていう名前で以下統一します。



使い方

使いたい鯖の適当なテキストチャンネルで以下のコマンドをチャットで発言

「?listener」発言を聞く側を今いるボイスチャンネルに召喚する。
「?streamer」聞いた内容を流す側を今いるボイスチャンネルに召喚する。
「?bye」botを切断させる。
「?volume @<ユーザータグ> <0-100の数字>」@ユーザーの音量を0-100の間で調整する。

動かなくなったら再起動で多分直ると思う。
